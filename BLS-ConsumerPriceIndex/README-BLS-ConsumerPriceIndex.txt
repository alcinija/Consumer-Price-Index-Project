This folder has US Govt data on the Consumer Price Index

Documentation files in this folder:

1. overview.txt - downloaded from
     https://download.bls.gov/pub/time.series/overview.txt
   That site indexes many different download folders as described in overview.txt.

2. BLS-cu-webindex.txt - snapshot of the webpage source for the contents of this folder.
   All the necessary files are here. Some others on that webpage are subsets.

3. cu.txt - detailed explanation of how the data files are interconnected.
